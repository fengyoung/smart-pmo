# pmo-burn-down — 燃尽图（P3 规划中）

> ⚠️ **状态：规划中**。本 Skill 为 P3 优先级，等待实施资源。

## 规划功能

- 选择目标里程碑，基于关联待办生成从计划日期到当前日期的燃尽曲线
- ASCII 终端渲染燃尽图（理想线 vs 实际线）
- 偏差告警：实际线高于理想线超过 20% 时标记 🔴
- 预计完成日期推算：基于当前完成速率外推

## 实现要点

1. 需要里程碑关联待办数据（`关联待办` link 字段）
2. 需要待办的创建时间和完成时间（当前 Base 设计已满足）
3. 燃尽算法：理想线 = 计划日期范围线性插值；实际线 = 每日累计完成计数值
4. 输出为 ANSI 终端图形或纯 ASCII
