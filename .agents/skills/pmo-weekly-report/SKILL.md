# pmo-weekly-report — 周报生成

## 执行方式

```bash
# 生成本周周报
claude pmo-weekly-report

# 生成指定周的周报（传入该周任意一天，自动计算周一到周日范围）
claude pmo-weekly-report --week YYYY-MM-DD
```

## 执行流程

1. 确定周范围：
   - 默认：本周周一 ~ 本周周日（ISO 周，周一为第一天）
   - `--week YYYY-MM-DD`：以该日期所在周的周一 ~ 周日为范围
2. 从 Base 收集数据：
   - 本周会议记录（会议日期在周范围内）
   - 本周新增待办（创建时间在周范围内）
   - 本周完成待办（完成日期在周范围内）
   - 逾期待办（截止日期 < 今天 且 状态≠已完成/已取消）
   - 里程碑状态快照
   - **下周计划**：截止日期在下周一～下周日之间、状态≠已完成/已取消的待办，按优先级排序
3. AI 整理周报内容
4. 展示草稿给用户确认
5. 用户确认后：
   - 通过 `lark-doc` 创建周报文档
   - 通过 `lark-wiki` 归档到 `02-周报/`
   - 通过 `lark-im` 推送摘要到项目群

## 异常处理

| 场景 | 处理方式 |
|------|---------|
| 本周无会议 | 周报正常生成，会议部分显示"本周无会议" |
| 本周无新增/完成待办 | 显示 0，不影响周报生成 |
| 已生成过本周周报 | 提示"本周周报已存在，是否覆盖？[y/N]" |
