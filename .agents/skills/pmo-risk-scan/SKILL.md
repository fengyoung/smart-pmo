# pmo-risk-scan — 项目风险扫描

> 自动扫描所有关注项目，检测风险信号并输出风险等级，帮助 PM 快速识别需要关注的项目。

## 执行方式

```bash
# 扫描所有关注项目
claude pmo-risk-scan

# 扫描指定项目
claude pmo-risk-scan --project XRay

# 仅输出高风险项目（🟡 及以上）
claude pmo-risk-scan --high-only

# 生成风险报告并推送到项目群
claude pmo-risk-scan --send
```

## 前置条件

无。`--project` 模式不需要设置当前项目；默认模式需至少有关注项目或 active 项目。

**公共模式引用：**
- 📋 待处理队列检查详见 [`_shared/pending-queue-check.md`](../_shared/pending-queue-check.md)
- ⏱️ Base 查询超时遵循公共配置：单次 20s，并发 30s（见 CLAUDE.md）
- 🔄 所有飞书 API 写操作遵循公共错误重试策略：3 次指数退避重试（1s/3s/5s）
- 🔍 配置加载时执行 schemaVersion 检查和配置完整性校验（见 CLAUDE.md）

## 执行流程

### 第1步：确定目标项目

1. 若指定 `--project <项目名>` → 仅扫描该项目
2. 否则读取 `~/.smart-pmo/pinned` 获取关注项目列表
3. 若 pinned 为空 → 收集所有 `status=active` 的项目

### 第2步：并行采集风险指标

对每个目标项目，并行查询以下风险指标：

| 指标 | 查询方式 | 风险阈值 |
|------|---------|---------|
| 过期待办率 | 截止日期 < currentDate 且 状态≠已完成/已取消 / 全部待处理+进行中 | >30% 🔴 / >15% 🟡 |
| 严重过期待办 | 过期超过 14 天的待办数量 | ≥3 🔴 / ≥1 🟡 |
| 会议断层 | 最近一次会议日期距今天数 | >14天 🔴 / >7天 🟡 |
| 里程碑延期率 | 状态=已延期 / 全部非已完成里程碑 | >50% 🔴 / >0% 🟡 |
| 僵尸待办 | 创建超过 30 天且状态仍为待处理 | ≥5 🔴 / ≥2 🟡 |
| 无进行中待办 | 状态=进行中 的待办为 0 但有待处理待办 | 🟡 |

### 第3步：综合风险评级

每个项目按风险指标的最高级别评定：

```
🟢 健康   — 所有指标正常
🟡 关注   — 至少 1 项黄灯
🟠 警告   — 至少 1 项触及黄灯上限，或多项黄灯
🔴 危险   — 至少 1 项红灯
```

### 第4步：展示风险报告

```
🔍 项目风险扫描 · {today}
═══════════════════════════════
扫描 {N} 个项目

🔴 XRay（XRay拆修检测2026）— 危险
  · 过期待办率 42%（11/26）— 超过 30% 阈值
  · 严重过期 4 条（超 14 天未处理）
  · 僵尸待办 4 条（超 30 天无进展）
  → 建议：优先清理过期待办，对僵尸待办明确关闭或重新分配

🟡 RCA（工艺归因系统）— 关注
  · 会议断层 10 天（上次会议 06-04）
  · 里程碑延期率 50%（1/2 已延期）
  → 建议：本周内安排一次项目同步会议

🟢 数据平台V2 — 健康
  · 无风险信号

─────────────────────────────
使用 pmo-use <项目名> 切换到风险项目进行跟进
使用 pmo-risk-scan --send 将报告推送到各项目群
```

### --send 推送模式

1. 按项目分别生成风险卡片
2. 推送到各项目配置 `larkResources.chatIds` 的第一个群（仅对有风险的项目推送）
3. 健康项目不推送（减少噪音）
4. 推送前预览卡片内容

## 异常处理

| 场景 | 处理 |
|------|------|
| Base 查询超时 | 该项目标注 ⚠️ 跳过，不影响其他项目 |
| 某项目无待办/里程碑 | 对应指标显示 `N/A`，不纳入风险计算 |
| 某项目无会议记录 | 会议断层指标直接标 🟡（无法评估） |
| 全部项目健康 | 展示 `🎉 所有项目状态健康` |

## 边缘情况

| 场景 | 处理方式 |
|------|---------|
| 无关注项目且无 active 项目 | 提示「没有可扫描的项目」 |
| 新项目（创建不足 7 天）| 标注 🆕，放宽阈值（如过期容忍 50%） |
| archived 项目 | 默认不扫描，除非显式 --project 指定 |
