# pmo-retro — 复盘助手（P3 规划中）

> ⚠️ **状态：规划中**。本 Skill 为 P3 优先级，等待实施资源。

## 规划功能

- 从会议纪要中 AI 提取「问题」「改进」「风险」关键词
- 汇总过期待办的原因归类（负责人变更、依赖阻塞、需求变化等）
- 生成结构化复盘报告：做的好的 / 待改进的 / 行动计划
- 归档到知识库 `05-项目资料/`

## 实现要点

1. 需要读取知识库中会议纪要文档正文（`01-会议纪要/` 下的 docx）
2. AI 语义分析提取复盘要素（基于关键词 + 上下文）
3. 与 pmo-meeting-process 提取的「关键决策」字段联动
4. 典型使用场景：里程碑完成时、项目阶段结束时
